Update Log

This Update:

- Feature Rollback: Rolled back to using Linear texture interpolation.


Thank you for your valuable feedback! I initially made Dripzels because I didn't like that ingame.clothing uses a light theme lol. I just wanted to make a more user friendly website while practicing my programming skills, but it has grown to be helpful to many clothing designers, so every feedback counts.


I have read every feedback suggestions and I'll be adding most of it in Dripzels 3.0 update. Some of the suggestions I like and will prossibly add:

  • Advanced clothing template to Roblox standard template converter
  • More Robux Calculator Options.
  • Option to hide each individual part of the humanoid
  • Will be switching to only R15 humanoids. Adding R6 is more time consuming and the preview is not as accurate either for it. I'll try adding some R6 humanoids after the initial 3.0 update but no promises.
  • More 2D Preview Styles.
  • "The ability to add on accessories onto the model via a link" by Chaos. I like this idea! Sadly, Roblox has a rate limit to their API now so I cannot add this. I will however add a feature to change the humanoid's face.
  • A toggle to switch between Linear (Blurry but Roblox accurate) and Closest (Pixelated) interpolation.
  • Will add a drag and drop feature and a donation page, thank you OOX1R. The first few donations will go towards owning the domain for the next few years ðŸ’œ
  • Regarding skin tones, this feature has been requested for a while now, but my old code was very messy so I didn't feel like going through it. Since I'm rewriting everything I'll try to add this.
  • Poses for rigs? Very less likely but I'll give it a try.

This might take a few months to finish since I want the update to be polished. If you would life to give more feedback you may do so here

Upcoming Updates:

I have been rewriting the whole website for a while now, the next update will add some features, fix some known bugs, improve perfomance, and improve the UI. Expect this to take some time as I'm the only developer working on the website. - Dori7os
